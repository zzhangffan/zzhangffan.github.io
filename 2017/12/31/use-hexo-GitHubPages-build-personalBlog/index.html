使用hexo+GitHub Pages搭建个人博客

12月 31 2017

准备工具

  • Node.js（必须）用于支持hexo，生成静态页面
  • Git（必须）用于推送本地内容
  • 注册一个GitHib账号（必须）存放文件，服务器

搭建环境

  1. Node.js

    Windows

    官网下载.msi文件到本地，双击安装即可

    Linux(CentOS)

    使用命令sudo yum install nodejs安装Node.js

    需要下载安装一系列依赖，输入y后回车，安装完成后，使用命令node -v检验安装版本
    我的输出如图，出现类似信息即可

  2. Git

    Windows

    官网下载安装软件包到本地，双击安装即可

    Linux(CentOS)

    参考：如何在CentOS 7中安装Git

  3. GitHub

    去注册一个GitHub账号 填写示例如下图：

    填写用户名、邮箱和密码

    选择个人计划，默认就行

    注册完整进入首页如图后

  4. 创建一个仓库做GitHub Pages（What is GitHub Pages?）

    进入刚注册的GitHub账户首页后，点击红圈处按钮


    提示请验证邮箱，刚紧去你的邮箱看看吧，打开邮件点击红框中链接进行验证

    跳转如下页面，填写相关信息

    点击创建仓库

    现在可以用你刚建立的仓库名称访问你的GitHub Pages了
